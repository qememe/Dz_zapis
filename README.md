# 📘 DZ Записная книжка

Красивое и простое веб-приложение в **одном файле `index.html`** для ведения домашних заданий.

## ✨ Возможности

- Добавление ДЗ с указанием:
  - **предмета**
  - **даты**
  - **текста задания**
  - **изображения** (опционально)
- Хранение записей в браузере через `localStorage`
- Фильтрация заданий по дате
- Экспорт всех заданий выбранной даты в **PDF**
- Современный адаптивный интерфейс на русском языке

---

## 🚀 Быстрый старт

### Вариант 1: просто открыть файл

Откройте `index.html` в браузере двойным кликом.

### Вариант 2: через локальный сервер (рекомендуется)

```bash
python3 -m http.server 8000
```

После запуска откройте:

- `http://127.0.0.1:8000/index.html`

---

## 🧭 Как пользоваться

1. В блоке **«Новое задание»** заполните предмет и дату.
2. Добавьте текст ДЗ и/или прикрепите изображение.
3. Нажмите **«Сохранить задание»**.
4. Чтобы выгрузить задания за конкретный день:
   - выберите дату в поле **«Фильтр по дате»**
   - нажмите **«Экспорт в PDF по дате»**.

## 📌 Примечания

- Данные сохраняются локально в вашем браузере (`localStorage`).
- При очистке данных браузера записи могут быть удалены.
- Для экспорта PDF требуется интернет при первом открытии страницы, чтобы загрузить библиотеку pdfmake с CDN.
